# Django在线商城管理系统 python327

## 项目概述

Django在线商城管理系统是基于Python的Django框架开发的，适用于构建和管理在线电子商务平台。系统采用前后端分离的设计模式，前端展现与后端逻辑处理相分离，保证了系统的可维护性和扩展性。

## 技术栈

### 前端技术

- JavaScript
- CSS
- HTML

### 后端技术

- Django
- MySql
- Python

## 功能模块

1. 用户管理
2. 商品管理
3. 订单管理
4. 营销活动
5. 数据统计与分析

## 系统角色

- 管理员：负责系统管理和维护
- 商家：负责商品上架、订单处理等
- 买家：浏览商品、下单购买

## 运行环境

- 操作系统：Linux或Windows
- Python版本：3.x
- 数据库：MySql

## 部署步骤

1. 安装Python环境
2. 安装MySql数据库
3. 克隆项目代码
4. 导入数据库结构
5. 运行项目
